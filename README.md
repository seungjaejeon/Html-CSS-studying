# CSS

<<style>>

h1 {
     font-size: 64px; 폰트 사이즈
     
     text-align: center; 중앙으로 놓기
     
     margin-top: 100px; 위로 100px 띄기
     
}
</style>>

(body) page에 나타나는 내용을 감싸주어야함

(head) 제목 CSS 자바스크립트 등 내용외의 많은 내용을 감싸줌

(html) 모든 코드를 감싸주면 됨

# 링크걸기
